Сайт предназначен для парка аттракционов

Инструкция запуска:
1 - Установить библиотеки(если нету) через requirements.txt
2 - Запуск сайта через main.py

Описание:
<------------------>
Данный сайт предназначен для сбора информации о парке и его аттракционов.
Удобный и совеременный дизайн сайта.
Выберите для себя на каком аттракционе вы бы захотели покататьтся.
Доброжелательный персонал парка поможет вам разобраться в парке, и, конечно,
низкие цены и хорошее качество только у нас!

Здесь вы можете:
1 - Зарегестироваться и авторизоваться
2 - Пополнить баланс для услуг
3 - Совершить онлайн покупку билетов на аттракционы
4 - Узнать о каждом аттракционе в парке
